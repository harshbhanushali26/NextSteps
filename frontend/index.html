The Process

Five phases.
One agent.

From raw PDF to interview-ready in under 2 minutes. Every phase hands off to the next automatically.

01

Parse & Extract

Upload your CV PDF and paste a job URL. Tavily fetches the full JD from any job board. Groq extracts structured profiles from both in seconds.

02

Skill Match

ChromaDB + cosine similarity scores every skill. See your exact match %, what you're strong on, and which gaps are blocking you.

03

Apply Package

Groq rewrites your resume bullets for the specific role. Cover letter streams out enriched with real company intelligence from Tavily.

04

Mock Interview

10 role-specific questions generated from your gap report. Every answer scored on 3 axes against your CV via RAG. Final performance report.

05

AI Tutor

Socratic tutor that teaches every gap skill without lecturing. Tavily enriches lessons with current examples. Code coach with 4-axis review.

Under the Hood

Built on production-grade
AI architecture.

No wrappers. No hype. Real agents with real infrastructure running every phase.

🧠

Groq + gpt-oss-120b

Every LLM call runs through Groq's inference engine on gpt-oss-120b. Sub-second structured extraction for CV parsing, bullet tailoring, cover letter generation and interview scoring.

→ 5 agent functions
🔍

Tavily Web Intelligence

Tavily extract fetches full JD text from any job board. Tavily search enriches cover letters with live company intelligence. Skill tutor context sourced from the real web.

→ 3 integration points
🧮

ChromaDB + RAG

Your CV is chunked and embedded into ChromaDB with SentenceTransformers. Cosine similarity powers skill matching. Interview answers are scored RAG-grounded against your actual experience.

→ all-MiniLM-L6-v2
⚡

FastAPI Layer

9 typed REST endpoints wrap every agent function with zero changes to agent logic. CORS-ready for the React frontend. Upload PDF, stream cover letter, run interview — all via clean JSON routes.

→ Railway deployed
⚛️

React + Vite Frontend

Phase-by-phase UI with drag-and-drop upload, streaming cover letter generation, chat-style interview simulator, Monaco code editor for practice problems, and radar chart performance summaries.

→ Vercel deployed
🛡️

Graceful Degradation

Every external API call is wrapped in try/except returning empty string on failure. Tavily unavailable? bs4 fallback scrapes the JD silently. The system never breaks — it degrades gracefully.

→ always-on design
